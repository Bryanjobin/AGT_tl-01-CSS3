A

Animation

Comme son nom l'indique, la propriété animation vient ajouter une animation de notre choix à un élément de notre choix ! Avec @Keyframes, nous donnons un nom de notre choix à l'animation, nous y ajoutons les propriétés CSS que nous voulons changer avec celle-ci, le temps d'action ainsi que le type d'animation.

Dans cet exemple ci-dessous, si nous déplaçons notre curseur sur la boîte orange, une animation viendra la changer de couleur et lui faire une rotation.

Placer le curseur ici

                            
                                
                                    .box {
                                        background-color: rgb(247, 148, 42);
                                        height: 100px;
                                        width: 100px;
                                    }

                                    /* Quand la souris sera en hover sur la boîte l'animation se déclenchera */
                                    .box:hover {
                                        animation: exemple_Animation 5s;
                                    }


                                    @keyframes exemple_Animation {
                                        from {
                                            background-color: rgb(247, 148, 42);
                                            transform: rotate(0deg);
                                        }
                                    
                                        to {
                                            background-color: cornflowerblue;
                                            transform: rotate(180deg);
                                        }
                                    }

B

Border-radius

Avec la propriété Border-radius, nous pouvons arrondir les coins d'éléments HTML comme ceux d'une photo par exemple.

Dans cet exemple ci-dessous, j'ai appliqué la propriété border-radius avec la valeur 8px sur une image. 

                            
                                
                                    img {
                                        border-radius: 8px;
                                    }

Box-shadow

Avec la propriété Box-shadow, nous pouvons ajouter une ombre à des éléments HTML.

Dans cet exemple ci-dessous, j'ai appliqué la propriété box-shadow avec les valeurs 5px(x), 12px(y) ainsi que la couleur cornflowerblue sur la boîte orange. 

                            
                                
                                    box {
                                        background-color: rgb(247, 148, 42);
                                        box-shadow: 5px 12px cornflowerblue;
                                    }

Caret-color

Avec la propriété Caret-color, nous pouvons changer la couleur du curseur nous indiquant où écrire du texte dans un élément HTML.

Dans cet exemple ci-dessous, j'ai appliqué la propriété caret-color et la valeur red sur une balise HTML input. Le curseur sera donc rouge. 

                            
                                
                                    input {
                                        caret-color: red;
                                    }

F

Filter

Avec la propriété CSS3 Filter, nous pouvons appliquer des filtres à des éléments HTML.

Voici une liste des valeurs possibles avec la propriété CSS filter :

  • none
  • blur()
  • brightness()
  • contrast()
  • drop-shadow()
  • grayscale()
  • hue-rotate()
  • invert()
  • opacity()
  • saturate()
  • sepia()
  • url()

Sur les trois images ci-dessous, la première contient la propriété filter avec la valeur brightness à 20%, la seconde contrast à 200% ainsi que la troisième saturate à 300%. 

                            
                                
                                    .image1 {
                                        filter: brightness(20%);
                                    }
                                    
                                    .image2 {
                                        filter: contrast(200%);
                                    }
                                    
                                    .image3 {
                                        filter: saturate(300%);
                                    }

Flex

La propriété Flex est une propriété raccourcie, elle regroupe les propriétés flex-basis, flex-grow et flex-shrink. Cette propriété peut contenir jusqu'à trois valeurs, une pour flex-basis, une pour flex-gros ainsi qu'une pour flex-shrink.

Placer votre curseur sur la boîte de couleur bleue pour voir la propriété flex faire son travail sur ses enfants. La boite rétrécira, seule la boîte 3 ne contient pas la propriété flex.

                                    .boite1 {
                                        flex: 20%;
                                    }
                                    
                                    .boite2{
                                        flex: 0 1;
                                    }
                                    
                                    .boite4 {
                                        flex: 1;
                                    }

G

Grid

Comme son nom l'indique, la propriété Grid vient ajouter une grille au CSS, cette grille permet entre autres de jouer avec la disposition des éléments. Grid est composé de colonnes (columns) et de rangées (rows) et d'une grille explicite et implicite. Afin de déterminé le nombre sur combien de row et de columns nous voulons disposer nos éléments, nous utilisons l'unité relative fr.

Dans cet exemple les boîtes orange seront disposées à l'aide de la propriété Grid ainsi que ses autres propriétés lui permettant une meilleure précision.

1

2

3

                            
                                
                                    .containeur {
                                        display: grid; 
                                        grid-template: 1fr 1fr 1fr / 1fr 1fr 1fr 1fr 1fr;
                                        gap: 0px;
                                        height: 100%;
                                    }

                                    .square1 {
                                        grid-row: 1 / 2;
                                        grid-column: 3 / 4;
                                    }

                                    .square2 {
                                        grid-row: 2 / 3;
                                        grid-column: 1 / 2;
                                    }

                                    .square3 {
                                        grid-row: 3 / 4;
                                        grid-column: 4 / 5;
                                    }

M

Media queries

media queries permet de modifier des propriétés css d'une page si des conditions sont remplies. Lorsque ces fameuses conditions sont remplies des valeurs css s'activent.

Voici une liste de caractéristiques sur lesquels les media queries peuvent se reposer

  • width
  • height
  • aspect-ration
  • orientation
  • resolution
  • scan
  • grid
  • update
  • overflow-block
  • overflow-inline
  • color
  • color-gamut
  • color-index
  • display-moded
  • monochrome
  • inverted-colors
  • pointer
  • hover
  • any-pointer
  • any-hover
  • light-level
  • prefers-reduced-motion
  • device-width
  • device-height

Si vous passez votre navigateur en mode fullscreen la couleur du texte de la phrase ci-dessous deviendra vert.

Incroyable n'est-ce pas?

                            
                                
                                    p {
                                        color: red;    
                                    }
                                    
                                    @media all and (display-mode: fullscreen) {
                                        p {
                                            color: green;    
                                        }
                                    }

Mix-blend-mode

Avec la propriété mix-blend-mode, nous pouvons déterminer de quelle façon l'élément viser va se mélanger avec son élément parent et son fond. Un peu comme les modes de lumières sur photoshop.

Il existe plusieurs valeurs attachées à cette propriété, en voici quelques-unes:

  • normal
  • multiply
  • screen
  • overlay
  • darken
  • lighten
  • color-dodge
  • color-burn
  • hard-light
  • soft-light
  • difference
  • exclusion
  • hue
  • saturation
  • color
  • luminosity

Dans cet exemple ci-dessous, vous pouvez apercevoir une photo de Ye qui marche avec ses grosses bottes avec la propriété mix-blend-mode ainsi que la valeur différence par-dessus une image gif. 

                            
                                
                                    .parent {
                                        align-items: center;
                                        background-image: url(https://media0.giphy.com/media/oe33xf3B50fsc/giphy.gif);
                                        display: flex;
                                        height: 200px;
                                        justify-content: center;
                                        width: 200px;
                                    }
                                    
                                    .ye {
                                        height: 150px;
                                        mix-blend-mode: difference;
                                    }

O

Opacity

Comme son nom l'indique, avec la propriété opacity nous pouvons déterminer l'opacité d'un élément html. La valeur doit être indiquée en %.

Dans l'exemple ci-dessous, l'image à la propriété css opacity accompagné de sa valeur à 40%. 

                            
                                
                                    img {
                                        opacity: 40%;
                                    }

T

Text-decoration-color

Dans un texte, la propriété CSS text-decoration-color sert à définir la couleur de la décoration du texte ajouté grâce à la propriété raccourcie text-decoration.

Dans l'exemple ci-dessous la phrase est sous-lignée en rouge avec l'ajout de la propriété CSS Text-decoration-color et de la valeur red.

Ceci est une phrase courte mais réussie !

                            
                                
                                    p {
                                        text-decoration:underline;
                                        text-decoration-color: red;
                                    }

Text-decoration-line

Avec la propriété CSS text-decoration-line, nous pouvons ajouter de la décoration linéaire dans les textes HTML.

Voici une liste des valeurs possible avec text-decoration-line:

  • none
  • underline
  • overline
  • line-through

Dans l'exemple ci-dessous, le texte est accompagné de la propriété CSS text-decoration-line ainsi que de la valeur line-through.

Voici une autre phrase courte mais réussie !

                            
                                
                                    p {
                                        text-decoration-line: line-through;
                                    }

Text-decoration-style

Avec la propriété CSS text-decoration-style nous pouvons intervenir sur le style de ligne ajouté avec la propriété text-decoration-line.

Dans cette phrase ci-dessous, nous venons ajouter la propriété text-decoration-line accompagnée par la propriété text-decoration-line et la valeur wavy.

Voici encore une autre phrase courte mais réussie !

                            
                                
                                    p {
                                        text-decoration-line: underline;
                                        text-decoration-style: wavy;
                                    }

Transform

Avec la propriété transform nous pouvons modifier la forme et la présentation d'un élément, comme le pivoter ou le déformer.

Voici une liste des valeurs possible avec cettre propriété

  • matrix
  • perspective
  • rotate
  • translate
  • scale
  • skew

Dans cet exemple ci-dessous, vous pouver voir plusieurs belles formes déformés ou déplacés par la propriétée transform. 

                            
                                
                                    #carre {
                                        transform: transform: translate(120px, 20px) scale(3, 1);
                                    }
                                    
                                    #etoile {
                                        transform: scale(1,0.5);
                                    }
                                    
                                    #pentagone {
                                        transform: skew(1deg, 20deg);
                                    }
                                    
                                    #para {
                                        transform: matrix(0.7,0.2,0.3,1,0.8,0.2);
                                    }
                                    
                                    #triangle {
                                        transform: rotate(0.2turn);
                                    }
                                    
                                    #cercle {
                                        transform: perspective(20px);
                                    }

Transition

Avec la propriété CSS transition, nous pouvons venir intervenir non seulement sur les animations mais également sur les changements d'état des éléments visés, permettant un changement plus fluide.

Dans cet exemple si on place le curseur sur le carré, il changera de couleur et deviendra également un rond avec la propriété CSS Transition et les valeurs background-color et border-radius à .5s . 

                            
                                
                                    .square {
                                        background-color: orange;
                                        width: 200px;
                                        height: 200px;
                                    }
                                    
                                    .square:hover {
                                        border-radius: 100px;
                                        background-color: cornflowerblue;
                                        transition: border-radius 5s, background-color 5s ;
                                    }
